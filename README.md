Pupil API
===============================================================================

A message and conversation HTTP API for Pupil



Requirements
-------------------------------------------------------------------------------

- MongoDB
- Node.js



Installation
-------------------------------------------------------------------------------

```bash
$ npm install
```



Usage
-------------------------------------------------------------------------------

```bash
$ MONGODB_URL=mongodb://mymongodb.com:27017 node app.js
```

You can omit the `MONGODB_URL` variable if you wish to use `localhost`.



Docker
-------------------------------------------------------------------------------

```bash
$ docker-compose build
$ docker-compose up
```
